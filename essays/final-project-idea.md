## Project: Language Library

### Overview
_The problem:_ Many students at UHM have to take a second language class as a requirement for graduation. While some languages may have a multitude of online sources that can be used to translate, other languages are not easily accessible or have inaccurate translations online. Some students may also just be curious about learning a new language, without having to take a course.

_The solution:_ Language Library is an application for UHM students to get translations or help with a language course that is offered at UHM. Users can also use this to make friends while learning a new langauge.

### Approach
The goal of this site is to help second language learners with the process of studying by providing easy access to online resources.
The possible roles in this system would be users and admins.


### Mockup page ideas
Some possible mockup pages include:
* Landing page
* User home page
* Directory with a list of languages
* Language pages
  * Search bar for searching up words in English or a second language
  * Page where users can add an entry to an ongoing online dictionary for that langauge
* StackOverflow like page where users can ask or answer questions with tags
* Admin home page
* Profile page
  * Displays languages that a user knows

### Use case ideas
* New user goes to landing page, reads how to find the language of their choice, also can log in and set up profile to ask or answer a question.
* Admin goes to landing page, logs in, gets home page, edits site for bad entries or questions
* User goes to specific language page to look up word in their choice of English or said language
* User can go to questions page where they can get help for things they need translation/clarification for.

### Beyond the basics
* Allow students and professors to rate and comment on the translations/dictionary entry, based on if it is a good/correct translation
* Creates a community outside of classroom where students can interact with each other, and with the professor
